# BT24 default plot settings.

Default plot settings for lineplots in the BT_22 for chapter 4.

## Usage

``` r
lineplot_4x4
```

## Format

### `lineplot_4x4`

A list with two elements.

- n_cols:

  Four columns of smaller plots are used for the final lineplot.

- axis_x_label_centralize:

  The x-axis labels are nudged into the center by 0.155 npc's.

- y_axis:
