# Set colours as cmyk values.

Set colours as cmyk values.

## Usage

``` r
cmyk(C, M, Y, K)
```

## Arguments

- C:

  Numeric cyan value.

- M:

  Numeric magenta value.

- Y:

  Numeric yellow value.

- K:

  Numeric key (black) value.

## Value

RGB colour value.

## Examples

``` r
cmyk(0, 0, 0, 1)
#> [1] "#FDFDFD"
```
