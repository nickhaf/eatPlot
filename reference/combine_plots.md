# Combine different ggplots

Combine different ggplots

## Usage

``` r
combine_plots(plot_list, plot_widths = NULL)
```

## Arguments

- plot_list:

  List of the plots that should be combined.

- plot_widths:

  Set the widths of the plots manually. Defaults to `NULL`, in which
  case the width will be calculated so the x axis scales are on the same
  metric.

## Value

Patchwork plot.
